Transformacin de la Escala del tem 4

Para garantizar que la escala de respuesta sea homognea en todo el cuestionario, se requiere que todos los tems tengan la misma direccin, es decir, que 1 represente menor felicidad y 7 mayor felicidad. Sin embargo, el tem 4 est invertido en comparacin con los otros tems, ya que en este caso:

  • 1 indica "muy feliz"
  • 7 indica "poco feliz" Para corregir esta inconsistencia, se realiz una transformacin en la que se invierte la puntuacin del tem 4 utilizando la siguiente frmula:

nuevo_valor = 8 valor_original

De esta manera:

  • Si un encuestado punta 1, el nuevo valor ser 8 1 = 7 8 1 = 7 , indicando mayor felicidad.
  • Si punta 7, se transformar a 8 7 = 1

def explorar_variables(data, columnas):

    resultados = {}

    for col in columnas:
        x = data[col]
        
        # Tamao de la muestra
        n = len(x)
        
        # Estadsticos bsicos
        media = np.mean(x)
        mediana = np.median(x)
        varianza = np.var(x, ddof=1)     # ddof=1 para varianza muestral
        std = np.std(x, ddof=1)
        minimo = np.min(x)
        maximo = np.max(x)
        rango = maximo - minimo
        iqr = stats.iqr(x)              # rango intercuartlico
        asimetria = stats.skew(x)
        curtosis = stats.kurtosis(x)
        
        # Media recortada al 5%
        media_recortada_5 = stats.trim_mean(x, 0.05)
        
        # Intervalo de confianza 95% para la media (suponiendo distribucin ~ normal)
        sem = std / np.sqrt(n)  # error estndar
        ci_95 = stats.t.interval(0.95, n - 1, loc=media, scale=sem)
        ci_inferior, ci_superior = ci_95

        # Guardamos los resultados en un diccionario
        resultados[col] = {
            #'count': n,
            'media': media,
            'IC_95%_inf': ci_inferior,
            'IC_95%_sup': ci_superior,
            'media_recortada_5%': media_recortada_5,
            'mediana': mediana,
            'varianza': varianza,
            'desv_estandar': std,
            'minimo': minimo,
            'maximo': maximo,
            'rango': rango,
            'rango_intercuartil': iqr,
            'asimetria': asimetria,
            'curtosis': curtosis
        }
    
    # Convertimos el diccionario en un DataFrame para visualizarlo mejor
    resultados = pd.DataFrame(resultados).unstack()#.T
    return resultados
